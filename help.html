how to do animations (top secret)

http://www.flickgame.org/play.html?p=3f5390b0b2aca7bce1eb4a4f483e2354&a=1

(or if it's an exported html file, you can either 1) load it as mygame.html?a=1, or 2) edit the exported html file with a text-editor, look for the line 'if(urlVars["a"]==="1"){' and change it to just 'if (true){').
